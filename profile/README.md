# Duuka Protocol

> Permissionless franchise protocol. Deploy a node. Run an agent. Sell anything.

Duuka turns any physical vending machine into an autonomous on-chain business.
Each machine runs its own AI agent — managing marketing, inventory decisions, 
and treasury — while shareholders anywhere in the world can buy stock and earn 
from its revenue.

Built on Arbitrum. Payments via Rails.

---

## How It Works

**Node Operators** deploy a machine + agent in their location.
They set their starting inventory, configure their agent personality, 
and go live. The agent handles the rest.

**Shareholders** buy fractional stock in any running Duuka node.
No machine required. Just capital and a phone.

**Buyers** order from their phone — in class, at work, anywhere.
Pick up when ready. Pay with their local bank app.

## Stack

- **Chain:** Arbitrum One / Arbitrum Orbit (L3)
- **Contracts:** Solidity + Hardhat
- **Frontend:** Next.js
- **Agent:** Node.js
- **Payments:** Rails (local bank app integration)
- **Hardware:** Open source — 3D printable frame, custom PCB
